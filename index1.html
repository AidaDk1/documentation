Documentation of web application

Here, in this documentation you will learn how to make this web application using php,js,css,mysql. This web application is Restaurant management system. At the top there is a navigation bar with logo of restaurant and navigation menu. And at the top there is a banner which is the image of the foods. Then there are sections like about us (information about restaurant ), a categories(about food categories) foods section(Different types of foods) contact,and register sections where you can register to our system and order foods. Web app's background looks very nice because of angle shape and gradient color. In this web app i have 5 main folders.
1)Admin folder:In this folder i have created some files ,which enables admins to manage the system. Admin can add categories,foods,and delete them.
2)API:In this folder ,there is file called data.php,and it is connected to Mysql. If someone registers,logs in or orders food,admin can see them and manage them.
3)Backends: folder contains all files,which connects front end part with back end.
4)Chunks:This web applications design and front end part.
